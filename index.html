Conexión con Google Sheets:

  1. Crea un Google Sheets y ve a Extensiones > Apps Script.
  2. Pega este código: 
    function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var rowData = [
    e.parameter.vecesSalio,
    new Date(),
    e.parameter.grupo,
    e.parameter.nombres,
    e.parameter.apellidos
  ];
  
  for(var i=1; i<=50; i++) {
    if(e.parameter['pregunta'+i]) {
      rowData.push(e.parameter['pregunta'+i]);
    }
  }
  
  sheet.appendRow(rowData);
  return ContentService.createTextOutput("Éxito").setMimeType(ContentService.MimeType.TEXT);
}
  3. Ve a Implementar > Nueva implementación > Aplicación web.
  4. En "Acceso", pon Cualquier persona, autoriza y copia la URL.
  5. Pega esa URL en el archivo HTML, en const scriptURL = 'AQUI_VA_TU_URL';.
